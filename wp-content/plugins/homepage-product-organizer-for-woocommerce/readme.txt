Simple drag and drop solution for arranging the order products are displayed on a page.

== Description ==

Select the products you want to display on your homepage and organize their placement without having to learn any custom coding. Our drag and drop function allows you to easily arrange them in order from top-down and left-right. There is no limit to the number of custom product arrangements you can make and where you display them. Additional uses can be on Category pages, or on related product detail pages.

There is no easy method in WooCommerce to select specific products and feature them on the homepage or any desired page. The Homepage Product Organizer for WooCommerce provides the user with the ability to select any number of categories, and the products within them (all or some), and arrange them in the desired order of appearance top-down/left-right. when arrangement is finished and saved, the user can simply copy and paste the shortcode unique to that arrangement into any page on their WooCommerce store.

Your ability to maximize sales will greatly increase when you can hand pick and feature your best selling products on your homepage and control their exact placement. The Homepage Product Organizer Plugin for WooCommerce provides you with a quick and simple method to do this. This plugin gives you full control of product placement and works wherever you paste the shortcode.

= Features: =
*   Users can create multiple versions of custom organized products and put the shortcode wherever you want the selected products to appear.
*   Users create different shortcodes and use them at different times of the year if they sell seasonal items
*   Users can create unique categories like Specials, Closeouts, New Arrivals, Clearance Sale, or what ever you want to promote and feature them on the homepage or on their own page exclusively

The Homepage Product Organizer Plugin for WooCommerce creates a unique short code for each product arrangement that you create and save. Simply copy the shortcode and paste it into the page you want it to appear.

== Installation ==

Installation and uninstallation are extremely simple. You can use WordPress' automatic install or follow the manual instructions below.

= Installing: =

1. Download the package.
2. Extract it to the "plugins" folder of your WordPress directory.
3. In the Admin Panel, go to "Plugins" and activate it.

= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Go to the "plugins" folder of your WordPress directory and delete the files/folder for this plugin.

= Usage: =

Using this plugin is very simple. Once it has been activated, 'Product Organizer' will be visible in the menu (Lite version). Pro version will read "Woo Organizer" in menu.


Get support for this plugin by going to the WooPlugins Contact Us page (http://wooplugins.co/support/). The Lite Version is free to use and allows the User to create one custom arrangement. The Pro version allows the user to create an unlimited quantity of arrangements per website with multisite licensing options available for purchase.


== Frequently Asked Questions ==

= Where can I get support for the plugin? =

   Get support for this plugin by going to the WooPlugins Contact Us page (http://wooplugins.co/support/).

= How do I get the shortcode to work on my homepage? =

   Pages with php require an added code. If you use this php string, make sure you change the quotation marks to apostrophes within our shortcode on either side of the number. Specific examples can be seen on our User Guide page: http://wooplugins.co/manual/

= Why aren't all of the products in the category that I selected appearing on my page?

   The number of products is controlled by the Blog post field. To change this, go to Settings > Reading Settings > Blog pages show at most > enter the maximum number of products you want to appear. 


= How can I add custom css? =

   You can paste the custom css in the "Product Organizer > Settings" section on the menu. This way, if there is a theme update your custom css won't be affected.


= How can I create more arrangements on my website and also use it on other websites I develop? =

   Our Lite version is free to use but only allows one arrangement per website. We have a Pro version that allows an unlimited number of arrangements as well as license use for 1, 5, or 50 websites.
